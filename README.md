# CSVToMarkdown README

This is CSVToMarkdown, a simple extension to easly convert CSV files to Markdown table.

## Usages

To convert:
* Select the corresponding text to CSV
* Press `CTRL + SHIFT + P`
* Select `Convert CSV to Markdown Table`
* Done!
